# SomaShare
SomaShare is a university textbook exchange platform designed to help students buy and sell textbooks affordably.

# Features
- Add, edit, and delete textbooks
- Search textbooks
- Dashboard overview
- Offers system (basic implementation)
- Trust score system

# Technologies Used
- .NET 8
- Blazor Server
- SQLite
- Entity Framework Core

# How to Run
1. Open terminal in project folder
2. Run: dotnet build
3. Run: dotnet run
